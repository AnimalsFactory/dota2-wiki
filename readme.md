# Справочник Dota 2 Wiki

Учебный проект на Flask — полноценный MVP справочника по игре Dota 2.

## Функциональность
- Главная страница с карточками разделов
- 4 информационных раздела (История, Герои, Предметы, Механики)
- Гостевой личный кабинет (история посещений + заметки)
- Интерактивная викторина (мастер-класс)
- Сохранение данных в SQLite (между сессиями)
- Адаптивный дизайн
